# Bishop QA Checklist

A plugin is not done until these pass:

- [ ] CMake config succeeds.
- [ ] Release build succeeds.
- [ ] VST3 artifact is produced.
- [ ] Standalone artifact is produced when enabled.
- [ ] Plugin opens without crashing.
- [ ] Parameters automate and save/load state.
- [ ] Bypass/gain staging is safe.
- [ ] No clicks/pops from parameter changes.
- [ ] CPU use is reasonable.
- [ ] Version/product name is correct.
